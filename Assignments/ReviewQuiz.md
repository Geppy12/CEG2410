
Quiz on essential commands 


1. Which command is used to display the current directory you are working in?
A: pwd . 

2. How do you list all files in a directory, including hidden files (those starting with a dot)?
A: ls -a   

3. Which command is used to create a new, empty directory?
A: mkdir    

4. What is the correct syntax to move a file named 'file.txt' into a folder named 'backup'?
A: mv file.txt backup/  

5. Which command allows you to view the contents of a file directly in the terminal?
A: Cat

6. What does the 'cd ..' command do?
A: Moves you up one level to the parent directory.

7. Which command is used to delete a file permanently?
A: rm

8. How do you search for a specific string of text within a file?
A: grep 

9. Which command prefix allows you to run a command with administrative or root privileges?
A: sudo

10. What is the purpose of the 'man' command?
To display the manual page for a command.

11. Which command is used to clear the terminal screen of all previous output?

A: clear

12. What symbol is used to 'pipe' the output of one command into the input of another?

A: |

13. Which command allows you to change file permissions?

A: chmod

15. How do you stop a command currently running in the terminal?

A: Ctrl + C


Quiz on essential networking topics 


1. Which layer of the hierarchical network design model is responsible for providing high-speed transport between different distribution layer devices?

A: Core layer

2. What is the primary purpose of a subnet mask in an IPv4 network?

A: To distinguish the network portion of an IP address from the host portion.

3. A firewall that inspects the actual content of the data packets and understands specific protocols (like HTTP or FTP) is known as:

A: Application-Level Gateway

4. If you have a network address of 192.168.1.0 with a subnet mask of 255.255.255.192, how many usable host addresses are available in each subnet?

A: 62

5. In network design, what is the 'Demilitarized Zone' (DMZ) used for?

A: To host public-facing services while keeping them separate from the internal network.

6. Which firewall technology keeps track of the state of active connections and uses this information to determine which network packets to allow through?

A: Stateful Packet Inspection (SPI)

7. What is the CIDR notation for the subnet mask 255.255.240.0?

A: /20

8. Which network topology provides the highest level of redundancy but is also the most expensive to implement?

A: full mesh

9. In a firewall policy, what is the 'Implicit Deny' rule?

A: A rule that automatically drops any traffic not explicitly permitted by other rules.

10. Which of the following is a private IP address range as defined by RFC 1918?

A: 172.16.0.0 – 172.31.255.255

11. What is the primary benefit of implementing VLANs (Virtual Local Area Networks) in a network design?

A: To segment broadcast domains and improve security/performance.

12. When configuring a firewall, what is 'Egress Filtering'?

A: Filtering traffic as it leaves the internal network for the outside world.

13. Which subnet mask would you use to create at least 500 subnets from a Class A network address?

A: 255.255.128.0

14. In the Three-Tier Hierarchical Model, which layer is responsible for 'collapsing' multiple access layer switches and routing between VLANs?

A: Distribution Layer

15. What is the network address for the host 10.5.150.34 with a subnet mask of 255.255.255.224?

A: 10.5.150.32
